# Use Cases

## Paid Media Reporting

The project will help transform raw campaign exports into structured reports with key metrics, insights, and recommended actions.

## Campaign Optimization

The assistant will support the analysis of campaign performance across platforms such as Meta Ads, Google Ads, GA4, Shopify, and VTEX.

## Account Management Support

The project will help account managers prepare client updates, document campaign changes, and identify performance risks faster.

## Ecommerce Analysis

The assistant will support ecommerce teams by analyzing sessions, purchases, revenue, ROAS, conversion rate, and performance changes across different time periods.

## Insight Generation

The project will use a consistent structure for insights:

- Context
- Diagnosis
- Recommended action
