# Roadmap

## Phase 1: Marketing Workflow Research

- Identify repetitive paid media and reporting tasks.
- Define common inputs from Meta Ads, Google Ads, GA4, Shopify, and VTEX.
- Document recurring analysis frameworks used in campaign reviews.

## Phase 2: Prompt and Template Development

- Create reusable prompt templates for campaign analysis.
- Build reporting structures for weekly and monthly performance reviews.
- Define insight formats for account managers and performance teams.

## Phase 3: Prototype Development

- Create simple scripts to analyze CSV exports.
- Generate automated summaries from campaign data.
- Test recommendations based on changes in spend, ROAS, CTR, CPC, CPM, sessions, purchases, and revenue.

## Phase 4: Public Documentation

- Add examples.
- Improve documentation.
- Share workflows that other marketers can reuse.
